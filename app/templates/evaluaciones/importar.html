Importar evaluación

Volver a la biblioteca
Se creará una evaluación nueva a tu nombre. Si el título coincide con uno que ya tienes, no se reemplaza nada: se agrega una copia nueva, que luego puedes editar o eliminar.

Formato del archivo

El archivo debe ser un JSON con esta estructura:

{% raw %}{
  "formato": "fuenti.evaluacion",
  "version": 1,
  "titulo": "Seguridad en el laboratorio",
  "umbral_aprobacion": 60,
  "preguntas": [
    {
      "enunciado": "¿Qué elemento de protección personal es obligatorio?",
      "tipo": "opcion_multiple",
      "alternativas": [
        { "texto": "Gafas de seguridad", "es_correcta": true },
        { "texto": "Gorra", "es_correcta": false },
        { "texto": "Guantes de lana", "es_correcta": false }
      ]
    },
    {
      "enunciado": "El agua apaga un incendio de origen eléctrico.",
      "tipo": "verdadero_falso",
      "alternativas": [
        { "texto": "Verdadero", "es_correcta": false },
        { "texto": "Falso", "es_correcta": true }
      ]
    }
  ]
}{% endraw %}

Reglas para que el archivo sea válido

  • titulo: texto obligatorio (hasta 255 caracteres).
  • umbral_aprobacion: número entero de 0 a 100.
  • preguntas: debe haber al menos una.
  • Cada pregunta lleva enunciado y un tipo: "opcion_multiple" (de 2 a 6 alternativas) o "verdadero_falso" (exactamente 2).
  • En cada pregunta, una sola alternativa tiene "es_correcta": true.
  • En las verdadero_falso, el texto lo fija la app por orden (la primera alternativa es "Verdadero" y la segunda "Falso"); lo que importa es el orden y cuál marcas como correcta.
