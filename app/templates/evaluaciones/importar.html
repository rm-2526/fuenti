Pega aquí el JSON con las preguntas (por ejemplo, el que te generó una IA con el prompt de más abajo). El título y el umbral son los que escribes arriba. Se creará una evaluación nueva a tu nombre: importar siempre agrega, nunca reemplaza (aunque el título coincida con uno que ya tengas).

Formato del JSON

El JSON debe contener solo las preguntas, con esta estructura:

{% raw %}{
  "preguntas": [
    {
      "enunciado": "¿Qué elemento de protección personal es obligatorio?",
      "tipo": "opcion_multiple",
      "alternativas": [
        { "texto": "Gafas de seguridad", "es_correcta": true },
        { "texto": "Gorra", "es_correcta": false },
        { "texto": "Guantes de lana", "es_correcta": false }
      ]
    },
    {
      "enunciado": "El agua apaga un incendio de origen eléctrico.",
      "tipo": "verdadero_falso",
      "alternativas": [
        { "texto": "Verdadero", "es_correcta": false },
        { "texto": "Falso", "es_correcta": true }
      ]
    }
  ]
}{% endraw %}

Reglas para que el JSON sea válido

  • El título y el umbral se escriben en el formulario de arriba, no en el JSON.
  • El JSON lleva una lista preguntas con al menos una pregunta.
  • Cada pregunta lleva enunciado y un tipo: "opcion_multiple" (de 2 a 6 alternativas) o "verdadero_falso" (exactamente 2).
  • En cada pregunta, una sola alternativa tiene "es_correcta": true.
  • En las verdadero_falso, la primera alternativa representa "Verdadero" y la segunda "Falso" (el texto lo fija la app). Marca con "es_correcta": true la que sea la respuesta correcta: puede ser cualquiera de las dos.
